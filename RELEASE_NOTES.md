### Quick Start

1. Download the latest release
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** and select the `extension` folder
5. Configure your credentials in the extension settings

### âš™ï¸ Prerequisites

**Important:** Before using this extension, configure your 2FA settings:

1. Log in to the SSO system manually
2. Navigate to your 2FA settings
3. Set authentication method to **image matrix**
4. Select the **logo mark (triangle icon)** as your authentication image
5. Configure it to require **3 clicks** of the same image
